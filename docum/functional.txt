Common_user
1. получить список всех парковок - backend
2. наложение фильтров на список парковок - frontend
3. нажатие на имени парковки отображает новую страницу с полной информацией о ней и
 используемых на ней тарифах - backend
4. получение списка тарифов - backend
5. наложение фильтров на список тарифов - frontend
6. нажатие на имя тарифа отображает новую страницу с информацией о тарифе и всех парковках
 использующих данный тариф - backend

 Рабочий системы
1. просмотр текущего списка запаркованных автомобилей - backend
2. добавление нового занятого парковочного места со всей информацией - backend
3. удаление занятого парковочного места и выдача "чека" при удалении записи - backend
4. при нажатии на номер парковочного места отображается полная информация о месте и
 автомобиле с владельцем - backend

 Администратор
- просмотр списка всех парковок - backend
- просмотр списка всех парковок + удалённых(галочка на форме) - backend
- наложение фильтров на парковки - frontend
- добавление парковки - backend
- настройка парковки - тарифные планы, места и тд. - backend
- удаление парковки. При этом все занятые места на парковке автоматически закрываются
 и выдаются чеки на каждый из них. очищается связь с тарифами - backend
- просмотр списка всех тарифов - backend
- просмотр списка всех тарифов + удалённых - backend
- наложение фильтров на тарифы - frontend
- добавление тарифа - backend
- настройка тарифов - backend
- удаление тарифов. очищается связь с парковками - backend
- получение списка всех запаркованных авто - backend
- получение списка всех запарковынных авто + удалённых - backend
- фильтры и поиск по авто - frontend
